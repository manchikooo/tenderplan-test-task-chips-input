Для запуска проекта нужно: 
1. Прописать в консоль yarn install
2. Дождаться загрузки пакетов
3. Прописать в консоль yarn start

Откроется приложение, в котором по центру экрана находится chipsInput. 
При вводе текста чипсы создаются, когда мы ставим запятую. Запятая внутри кавычек не создает новый чипс. Чипсы удалются, если в них не остается текста или если мы, находясь на пустом инпуте ввода нового чипса нажимаем backspace. 
Не реализовано выделение нескольких чипсов и их удаление.
